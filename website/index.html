How it works

1

Add to Chrome

Install the free extension from the Chrome Web Store. No sign-up, no account.

2

Open WhatsApp Web

Head to web.whatsapp.com and open any chat or group you want help with.

3

Open the WACAO panel

Click the icon to slide out the side panel, then translate or summarize.

Requires Chrome 138+ with on-device AI enabled. WACAO walks you through the one-time setup on first launch.
